INTERACTIVE SALES DASHBOARD


1. Project Overview

 The Interactive Sales Dashboard is a data visualization project developed using Python to analyze sales data and present business insights through statistical and interactive visualizations. The project aims to transform raw sales data into meaningful charts that help understand sales performance, product popularity, customer behavior, and regional sales trends.
The dashboard uses Pandas for data manipulation, Seaborn and Matplotlib for statistical charts, and Plotly for interactive visualizations. The project also automatically saves all generated charts in a separate folder for documentation and future reference.

2. Project Objectives

 The primary objectives of this project are:
Load and analyze sales data from a CSV file.
Clean and prepare the dataset for analysis.
Visualize sales information using multiple chart types.
Create interactive charts with Plotly.
Generate business insights automatically.
Build a professional dashboard layout.
Save all visualizations for reporting and presentation.

3. Dataset Description

 The project uses a sales dataset named sales_data.csv.
The dataset contains the following columns:

Date

Product

Quantity

Price

Customer_ID

Region

Total_Sales

Each row represents a sales transaction including the purchased product, quantity sold, sales region, customer identification, and total sales amount.

5. Project Workflow

The project follows these steps:

Step 1: Import Libraries

The required Python libraries are imported to perform data analysis and visualization.

Step 2: Create Output Folder

A folder named visualizations is automatically created to store all generated charts.

Step 3: Load Dataset

The CSV file is loaded using Pandas.

The dataset is then displayed to verify that it has been imported successfully.

Step 4: Data Exploration

The program displays:

Dataset preview

Column names

Dataset dimensions

Missing values

Statistical summary

This helps understand the dataset before analysis.

Step 5: Data Preparation

The Date column is converted into DateTime format.
A new Month column is created from the Date column for monthly sales analysis.

Step 6: Visualization

Several visualizations are created to analyze sales performance.

6. Visualizations

6.1 Bar Chart

Title:

 Total Sales by Product
 
Purpose:

 Displays the total sales generated by each product.
 
Interpretation:

 The chart helps identify the best-selling and least-selling products.


6.2 Line Chart

Title:

 Monthly Sales Trend
 
Purpose:

 Shows how total sales change over different months.
 
Interpretation:

 The chart helps identify sales trends, seasonal demand, and peak sales periods.


6.3 Box Plot

Title:

 Sales Distribution by Region
 
Purpose:

 Shows the spread of sales across different regions.
 
Interpretation:

 The box plot displays the median, quartiles, and potential outliers in regional sales.


6.4 Violin Plot

Title:

 Product Sales Distribution
 
Purpose:

 Displays the distribution and density of sales for each product.
 
Interpretation:

 The violin plot provides a detailed view of how sales values are distributed.


6.5 Correlation Heatmap

Title:

 Correlation Heatmap
 
Purpose:

 Shows the relationship between Quantity, Price, and Total Sales.
 
Interpretation:

 A darker color indicates a stronger relationship between variables.


6.6 Scatter Plot

 Shows the relationship between product price and total sales.
 
Interpretation:

 Different colors represent different sales regions, making it easy to compare regional performance.
